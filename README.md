# Seoul Winter Vibe Trip

這是一個為首爾 9 天 8 夜冬季自由行設計的互動式行程網頁。

## 如何瀏覽

1.  直接用瀏覽器打開 `index.html` 檔案。
2.  或是使用 VS Code 的 "Live Server" 擴充功能來預覽。

## 特色

-   **Vibe Coding 風格**: 現代、簡約、冬季氛圍。
-   **互動式時間軸**: 隨著捲動淡入顯示行程。
-   **響應式設計**: 手機與電腦皆可瀏覽。
-   **詳細資訊**: 包含每日景點、交通、美食與預算。

## 檔案結構

-   `index.html`: 網頁主結構與內容。
-   `style.css`: 樣式設計與動畫效果。
-   `script.js`: 互動邏輯與捲動特效。
